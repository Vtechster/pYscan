# Project Title
Simple Python IP Scanner
## Description
• Scans IP for open ports<br>
• Returns Banner<br>
• 4 port range modes including custom range<br>
## Getting Started
### Dependencies
* Colorama
* Pyfiglet
### Installing
```
git clone https://github.com/Vtechster/pYscan
cd pYscan
chmod a+x main.py
pip3 install -r requirements.txt
python3 main.py
```
